---

La comunidad de Neverwinter Nights utiliza una terminología específica que es útil conocer al explorar módulos o mundos persistentes (PWs).

## Glosario Esencial

### Módulos y Contenido

| Término | Definición |
|----------|-------------|
| **Bicmups** | El nombre de archivo `.bic` para archivos de personaje. |
| **Módulos (Modules)** | El archivo principal `.mod` que contiene un mundo, historia y áreas asociadas. |
| **HAKs** | Archivos `.hak` que contienen recursos personalizados (texturas, modelos, scripts). |
| **TLK** | Archivos `.tlk` de conversación. Contienen el texto de los diálogos para localización. |
| **PW (Persistent World)** | Servidor multijugador 24/7 con progresión permanente, donde los personajes residen en el servidor. |

### Reglas y Roles

| Término | Definición |
|----------|-------------|
| **Metagaming** | Usar información fuera del juego para influir en tu personaje. Generalmente prohibido en PWs de rol estricto. |
| **Power-Gaming** | Maximizar el poder del personaje ignorando la narrativa o el rol. |
| **DM (Dungeon Master)** | Persona que usa el cliente de DM para narrar, controlar PNJs y eventos. |
| **RP (Roleplay)** | Interpretar a tu personaje según las reglas del servidor. Puede ser "RP Leve" o "RP Estricto". |

### Acrónimos de Combate y Desarrollo

| Acrónimo | Significado |
|-----------|-------------|
| **BBA** | Bonificador Base de Ataque (Base Attack Bonus). |
| **CA** | Clase de Armadura (Armor Class). |
| **CD** | Clase de Dificultad (Difficulty Class). |
| **STR, DEX, CON...** | Atributos principales del personaje. |
| **NWScript** | Lenguaje de scripting del Aurora Toolset. |

### Tipos de Módulos Famosos

- **Hack 'n Slash:** Enfocados en combate y obtención de botín.  
- **Story/Narrativa:** Basados en trama, diálogo y acertijos.  
- **Simulador Social:** PWs centrados en la vida cotidiana, con poco combate.
